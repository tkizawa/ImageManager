# Image Manager (画像・写真管理アプリ)

Image Manager は、指定したフォルダ内の画像ファイルを高速かつ簡単に閲覧・管理するための Windows 向けデスクトップアプリケーションです。
.NET 10 と WPF (Windows Presentation Foundation)、MVVM アーキテクチャを用いて構築されています。

## 主な機能

* **フォルダの選択**: PC 内の任意のフォルダを選択し、画像ファイル（.jpg, .png, .bmp, .gif など）を読み込むことができます。
* **サムネイル表示**: 選択したフォルダ内の画像が、自動的にサムネイル形式で一覧表示されます。
* **画像のプレビューと詳細情報の確認**: サムネイルをクリックして選択すると、右側のペインに拡大されたプレビュー画像と、以下のプロパティ情報が表示されます。
  * ファイル名
  * ファイルサイズ (Bytes)
  * 作成日時

## 使い方

1. **アプリの起動**
   * アプリケーションを実行して起動します。（開発環境の場合は `dotnet run` で起動可能です）
2. **画像の読み込み**
   * 画面左上の **「フォルダを開く...」** ボタンをクリックします。
   * フォルダ選択ダイアログが表示されるので、画像が保存されているフォルダを選択して「フォルダの選択」をクリックします。
3. **画像の閲覧**
   * 中央のリストに画像がサムネイル表示されます。
   * 見たい画像をクリックすると、右側に拡大画像とファイル情報が表示されます。

## 開発・ビルド環境

* **OS**: Windows 10 / 11
* **フレームワーク**: .NET 10.0
* **アーキテクチャ・ライブラリ**: 
  * WPF (Windows Presentation Foundation)
  * CommunityToolkit.Mvvm (MVVMパターンの実装に使用)

### 開発者向けセットアップ

リポジトリをクローンした後、以下のコマンドでビルドおよび実行が可能です。

```bash
# プロジェクトのルートディレクトリに移動
cd ImageManager

# アプリケーションの実行
dotnet run
```
