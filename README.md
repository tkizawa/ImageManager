# Image Manager (画像・写真管理アプリ)

Image Manager は、指定したフォルダ内の画像ファイルを高速かつ簡単に閲覧・管理するための Windows 向けデスクトップアプリケーションです。
.NET 10 と WinUI 3 (Windows App SDK)、MVVM アーキテクチャを用いて構築されています。

## 主な機能

* **フォルダツリーと選択**: 画面左側のツリービューからドライブやフォルダを階層的にたどり、画像ファイルを読み込むことができます。「フォルダを開く...」ボタンからの選択も可能です。
* **サムネイル表示とズーム**: 選択したフォルダ内の画像がサムネイル形式で一覧表示されます。`Ctrl + マウスホイール` でサムネイルサイズの拡大・縮小が可能です。
* **画像の右クリック操作**: サムネイルを右クリックするとコンテキストメニューが表示され、以下の操作が可能です。
  * **Windowsのフォトで開く**: 標準のフォトアプリで画像を開きます。
  * **エクスプローラーで表示**: 対象のファイルを選択した状態でWindowsのエクスプローラーを開きます。ここから右クリックでOneDrive標準の「リンクのコピー」などの操作が可能です。
* **画像のプレビューと詳細・EXIF情報の確認**: サムネイルをクリックすると、右側のペインにプレビュー画像と詳細プロパティが表示されます。
  * ファイル名、ファイルサイズ (Bytes)、ファイル作成日時
  * **EXIF情報**: 撮影日、カメラモデル、レンズ、F値、露出時間、ISO、焦点距離（※情報が含まれる写真のみ表示）
* **レイアウトのカスタマイズ**: フォルダツリー、サムネイル一覧、プレビュー欄の境界線をドラッグして幅を調整できます。調整した幅は自動保存され、次回起動時にも復元されます。

## 使い方

1. **アプリの起動**
   * アプリケーションを実行して起動します。（開発環境の場合は `dotnet run` で起動可能です）
2. **画像の読み込み**
   * 画面左上の **「フォルダを開く...」** ボタンをクリックするか、左側の **フォルダツリー** から任意のフォルダをクリックします。
   * 指定したフォルダ内の画像が読み込まれます（サブフォルダを開く状態も記憶されます）。
3. **画像の閲覧**
   * 中央のリストに画像がサムネイル表示されます。
   * 見たい画像をクリックすると、右側に拡大画像とファイル情報が表示されます。

## 開発・ビルド環境

* **OS**: Windows 10 / 11
* **フレームワーク**: .NET 10.0
* **アーキテクチャ・ライブラリ**: 
  * WinUI 3 / Windows App SDK (UIフレームワーク)
  * CommunityToolkit.Mvvm (MVVMパターンの実装)
  * MetadataExtractor (EXIF・メタデータ解析)

### 開発者向けセットアップ

リポジトリをクローンした後、以下のコマンドでビルドおよび実行が可能です。

```bash
# プロジェクトのルートディレクトリに移動
cd ImageManager

# アプリケーションの実行
dotnet run
```
